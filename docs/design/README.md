# Design references

These images are implementation references for the MeshMCP Agent OS design
system. Runtime UI remains semantic HTML/CSS/JavaScript; the screenshots are not
shipped as interface chrome.

- `agent-os-home-desktop.png` — desktop AppShell, Home hierarchy, and component anatomy.
- `agent-os-home-mobile.png` — mobile safe-area, horizontal Nearby rail, and bottom navigation.
- `air-universal-actions-desktop.png` — Universal Actions command palette, grouped search results, keyboard treatment, and policy-governed trust footer.
- `air-universal-actions-mobile.png` — service-aware Nearby action sheet, mobile safe-area behavior, and direct agent actions.

The Universal Actions references extend, rather than replace, the Agent OS
tokens in `../UX-AGENT-OS.md`: true white/light-blue surfaces, cobalt actions,
green verified state, hairline separators, system UI typography, 44px minimum
targets, and 150–320ms state motion. Interface text, controls, focus, selection,
search, and action results remain code-native.
